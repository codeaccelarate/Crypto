Cryptocurrency Project Explanation

Program Explanation

This program is a web-based application built using Flask that predicts cryptocurrency prices using LSTM neural networks. Here's a simplified breakdown:

  • The program uses the Binance API to retrieve historical price data of cryptocurrencies.
  • It preprocesses the data by extracting closing prices and normalizing them using MinMaxScaler.
  • The LSTM models (one for each cryptocurrency) are loaded with pre-trained weights.
  • When a user makes a prediction request through the web interface, the appropriate model is used to predict future prices.
  • Predictions are displayed along with real prices, percentage differences, and recommendations (buy, sell, hold) based on the prediction.
  • The program also generates plots comparing predicted prices with real prices for visualization.
  • The Flask framework enables the interaction through a user-friendly web interface.

Requirements

The code requires several libraries and tools to run successfully:

  • Flask for the web application framework.
  • Keras for building and training LSTM neural network models.
  • pandas for data manipulation and analysis.
  • numpy for numerical computations.
  • sklearn for data preprocessing (MinMaxScaler).
  • matplotlib for generating plots.
  • Binance API (`binance.client.Client`) for fetching historical cryptocurrency data.

How Data is Collected?

Data is collected using the Binance API (`binance.client.Client`). The code retrieves historical price data for specific cryptocurrencies (Bitcoin, Ethereum, Litecoin, Ripple) within a given time range. The data is returned as a DataFrame and then used for further analysis and prediction.

How Data is Filtered?

The historical price data fetched from Binance API is filtered to extract specific features, mainly the closing prices (`'Close'` column). These closing prices are used as the basis for predicting future prices.

What Algorithms are Used?

The primary algorithm used in this program is the Long Short-Term Memory (LSTM) neural network, implemented using Keras. LSTM networks are suitable for sequence prediction tasks, such as time series forecasting, which makes them ideal for predicting cryptocurrency prices based on historical data.

2. sklearn (from sklearn.preprocessing import MinMaxScaler)

MinMaxScaler is a preprocessing technique used to scale numeric data to a specific range, typically between 0 and 1. This technique is important for ensuring that all features contribute equally to machine learning models, especially when the features have different scales or units.

Key Points:

  • Scaling: MinMaxScaler performs scaling by shifting and rescaling features to a given range, often [0, 1]. This is achieved through the formula: X_scaled = (X - X_min) / (X_max - X_min), where X is the original feature, X_min is the minimum value of X, and X_max is the maximum value of X.
  • Normalization for ML Models: By scaling cryptocurrency prices using MinMaxScaler, we ensure that the price values are within a consistent range across different cryptocurrencies. This normalization aids in training machine learning models, such as regression models or neural networks, which can be sensitive to the scale of input features.
  • Usage in Predictive Models: In our project, MinMaxScaler is employed to preprocess cryptocurrency price data before feeding it into predictive models. Normalizing the price data helps improve model performance and stability during training and prediction phases.

Sample Code: Plotting


import io
import base64
import matplotlib.pyplot as plt

def plot_graph(crypto, predicted_prices, real_prices):
    plt.figure(figsize=(10, 6))
    plt.plot(predicted_prices, label='Predicted Price', color='orange')
    plt.plot(real_prices, label='Real Price', color='purple')
    plt.title(f'{crypto} Price Prediction')
    plt.xlabel('Timestamp')
    plt.ylabel('Price (USD)')
    plt.legend()
    
    # Save plot to in-memory buffer and encode as base64
    img_buffer = io.BytesIO()
    plt.savefig(img_buffer, format='png')
    img_buffer.seek(0)
    plot_url = base64.b64encode(img_buffer.getvalue()).decode()
    plt.close()
    
    return plot_url

from keras.models import Sequential
from keras.layers import Dense, LSTM

def model_definition():
    model = Sequential()
    model.add(LSTM(256, return_sequences=True, input_shape=(1, 1)))
    model.add(LSTM(256))
    model.add(Dense(1))

Machine Learning Model Used:

The script uses LSTM (Long Short-Term Memory) neural networks for predicting cryptocurrency prices. LSTM is a type of deep learning model particularly good at learning patterns in sequential data, like time series.

Algorithm:

  1. LSTM (Long Short-Term Memory):
    • LSTM is a type of neural network architecture suitable for sequence prediction tasks.
    • It's designed to capture long-term dependencies in data, which is important for time series forecasting like cryptocurrency prices.
    • The model consists of multiple LSTM layers followed by a dense (fully connected) layer for predicting the next value in a sequence based on historical data.

Steps in Machine Learning Part:

  1. Model Definition (`model_definition()`):
    • This function sets up the LSTM model architecture using Keras (a deep learning library).
    • It creates a neural network with two LSTM layers and one dense layer for prediction.
    • The model is configured to minimize prediction errors using the mean squared error loss function and the Adam optimizer.
  2. Loading Pre-Trained Models:
    • The script loads pre-trained LSTM models (`btc_model`, `eth_model`, etc.) from saved files (`btc_model.h5`, `eth_model.h5`, etc.).
    • These models have already been trained on historical cryptocurrency price data to make accurate predictions.
  3. Data Preparation (`data_extract()`):
    • Historical price data for a specific cryptocurrency (like Bitcoin or Ethereum) is fetched using the Binance API.
    • The data is processed and prepared to be fed into the LSTM model for prediction.
  4. Prediction (`predict()`):
    • When a prediction request is made (e.g., to predict Bitcoin's price for the next day), the appropriate LSTM model is selected based on the chosen cryptocurrency.
    • The model is used to predict the future price based on historical price data.
    • The predicted prices are then transformed back to their original scale to provide meaningful predictions.
